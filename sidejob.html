AI副业思路（零门槛，新手可做）

1. AI文案代写（最易上手，无成本）

适合人群：会用手机/电脑打字，有基础沟通能力的新手

所需工具：豆包AI、通义千问（免费）

操作步骤：

  1. 用豆包生成文案模板（小红书种草、朋友圈文案、短视频脚本、电商文案）；
  2. 在闲鱼、淘宝、小红书发布“AI文案代写”服务，标价10-50元/篇；
  3. 接单后，用AI生成初稿，人工简单优化（调整语气、贴合客户需求），交付客户；
  4. 积累客户后，可做包月服务（比如300元/月，帮客户写10篇文案）。

盈利预期：新手每天接3-5单，月入1000-3000元，熟练后可批量接单。

2. AI头像/海报设计（颜值高，易引流）

适合人群：对审美有基础要求，会用简单AI绘画工具的新手

所需工具：文心一格、Midjourney（免费+付费结合）

操作步骤：

  1. 用文心一格（免费次数）生成头像（古风、卡通、现代）、海报（活动海报、产品海报）；
  2. 在小红书、抖音发布作品（比如“AI头像定制，5元/张”），引流到微信接单；
  3. 单张头像收费5-20元，海报收费20-100元，定制款（比如情侣头像、企业海报）可提高价格；
  4. 批量制作热门风格头像，打包售卖（比如10张/29.9元），一次制作多次收益。

盈利预期：月入2000-5000元，引流效果好可翻倍。

3. AI资料整理/笔记代做（需求稳定，复购率高）

适合人群：学生党、宝妈，有耐心、会简单整理资料的新手

所需工具：讯飞听见（语音转文字）、豆包（整理优化）

操作步骤：

  1. 把客户提供的课程录音、视频，用讯飞听见转成文字；
  2. 用豆包把杂乱文字整理成结构化笔记（分章节、划重点）；
  3. 在闲鱼、淘宝发布“课程笔记整理”“资料代整理”服务，标价20-100元/份；
  4. 整理热门课程（比如考研、考公、技能课）的笔记，做成电子资料，挂网站/闲鱼自动售卖。

盈利预期：月入1500-4000元，电子资料可长期躺赚。

4. AI工具教学/代配置（信息差变现）

适合人群：熟悉各类AI工具，会简单教学的新手

所需工具：你网站收录的所有AI工具（豆包、ChatGPT、剪映AI等）

操作步骤：

  1. 在闲鱼、微信朋友圈发布“AI工具教学”“代注册、代配置”服务；
  2. 针对小白客户，一对一教学（比如“ChatGPT注册使用，39元/次”“剪映AI剪辑教学，59元/次”）；
  3. 帮客户配置AI工具（比如设置豆包提示词、优化Midjourney参数），收费30-100元/单；
  4. 结合你网站的教程，推出“AI工具全套教学”，收费99元/人，批量变现。

盈利预期：月入3000-8000元，越熟悉工具，收费越高。
